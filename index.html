Ransomware Detector

Utilising Machine Learning To Protect Our Most Important Assets

Learn More Watch The Demo Video
  • 1. Gather Data
    By Using Wireshark and TShark, The Team are Capturing Live Network Data While Also Using Publicly Available Network Captures
  • 2. Convert Data
    Once Data Has Been captured, The Team Convert The Captured Packets To CSV, Which Can then Be Read By The Machine Learning Environment, Weka
  • 3. Organise Data
    Once The Team Has All CSV Files Required, The Files Must Then Be Seperated Into "Ransomware" and "Non-Ransomware" Directories
  • 4. Train The Model
    We Now Train The Machine Learning Model By 'Feeding' The Model Our Data
  • 5. Test The Model
    Once We Have Trained The Model To Classify Data As Ransomware or Safe, We Now Test The Model With Unknown Data
